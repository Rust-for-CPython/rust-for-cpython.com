# Contributing

### Code

If you would like to contribute to Rust for CPython,
please check out our [fork of CPython](https://github.com/Rust-for-CPython/cpython).

We're currently focused on defining Rust APIs for implementing extensions in Rust. For more details please check out [the open issues](https://github.com/Rust-for-CPython/cpython/issues/).

### Weekly Meeting

We have a weekly meeting where we discuss the current work and future plans of the project. Details are available in the `#useful-links` channel [on Discord](./contact.md#discord).
